Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

Sticky is used when you scroll down and the pages still follow it, Mostly used for mobile phones.

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

I used pixels top and left to move around the realtive box. I used top 100 and 900 pixels left to move the box center of my screen.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

For top and right I used 100px right and top 90pxs. It affected it by moving it right and when you scroll down it stays there where I told it to be.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I used 0 values for bottom and right this made the pages exactly all the way at the bottom and all the way at the right. So it made it bottom right.
